# ChatMe — AI Website (Created by Akin S Sokpah — Liberia)

This repo contains:
- Frontend: React (Vite) + Tailwind + Firebase Google Sign-in.
- Backend: Node/Express proxy to OpenAI (keeps API key secret).
- 80 AI features (frontend triggers + backend endpoints).

**Important**
- Set `OPENAI_API_KEY` on Render (backend environment variables).
- Enable Google sign-in in Firebase and add your deployed frontend domain to authorized domains.
